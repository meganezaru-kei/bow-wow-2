# Bow Wow Album
犬の画像投稿アプリです。<br>
日々の成長記録等をアップして、ペットとの思い出を記録できます。<br>
レスポンシブ対応しているのでスマホからもご確認いただけます。

# URL
https://bow-wow-album.com/ <br>
ログイン画面の「かんたんログイン」をクリックすると、メールアドレスとパスワードを入力せずにログインできます。

# 使用技術
#### フロント
- Haml
- Scss
- jQuery
#### バックエンド
- Ruby 2.5.1
- Ruby on Rails 5.2.3
#### サーバー
- Nginx 1.16.1
#### DB
- MySQL 5.7
#### インフラ等
- Docker/docker-compose
- AWS（VPC, EC2, S3, Route 53, ALB, ACM）
- CircleCI（CI/CD)
- Capistrano3
- RSpec

- ユーザー機能
- 記事投稿機能
